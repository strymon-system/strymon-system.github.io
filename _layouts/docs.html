{% comment %} The following code creates the side navigation for the documentation pages. Documents with unexisting or misspelled categories, or no category at all, will be printed first by the first for loop below, i.e. under the title "Documentation". Any pages with valid pre-defined categories (i.e. categories defined in `docs_categories` collection the `_config.yml`) will be emitted in the according category in the second for loop. The order of categories is the same as the order defined in the `_config.yml`. Note: The `nav-index` of each document defines the order of the document within its category. Documents without any `nav-index` will be put at the bottom. {% endcomment %}
